# Time Series Forecasting: Stock Prices

## Overview
This group project explores time-dependent patterns in stock market data using
classical time series forecasting techniques. The objective was to compare
model performance and interpret forecasts in a business and financial
context.

## Methods
- Time series decomposition
- ARIMA modeling
- Exponential Smoothing
- Forecast comparison and evaluation

## Skills Demonstrated
- Time series analysis
- Forecasting and model comparison
- Quantitative reasoning
- Interpretation of uncertainty

## Tools
- Python
- Pandas
- NumPy
- statsmodels
- Matplotlib
